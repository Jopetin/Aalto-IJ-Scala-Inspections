Reports any calls to side-effect-free methods whose result is not stored or used further. This is to avoid mistakes where side-effect-free methods are mistaken as effectful. Side-effect-free methods are recognized from a custom annotation with the name configured below.

Example of wrong usage: 


class SideEffectFree extends StaticAnnotation

@SideEffectFree
def method(): Int =
    0

def main(): Int =
    method()
    0
