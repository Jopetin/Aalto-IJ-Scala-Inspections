Reports usage of get to fetch the value inside an Option wrapper.

The recommended solution is to use match or getOrElse.

Example of wrong usage: 


val x = Some("Hello world")

println(x.get)

The proper implementations: 


val x = Some("Hello world")

x match
    case Some(y) => println(y)
    case None => println("Not found")

println(x.getOrElse("Not Found"))
